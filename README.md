# nevelib-website

Public-facing website for `nevelib`, built as a separate Next.js repository.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- npm

## Local development

```bash
npm install
npm run dev
```

Production build:

```bash
npm run build
npm run start
```

## Routes

- `/`
- `/modules`
- `/architecture`
- `/getting-started`
- `/applications`
- `/publications`
- `/about`
